Minimal Radio für FiveM

Autor: Mopsy
Version: 1.0.0
Beschreibung:
Ein minimalistisches Radio-HUD für Fahrzeuge in FiveM.

Zeigt den aktuellen Radiosender im HUD an

Steuerung über Hotkeys (ein/aus, nächste/vorherige Station)

Nutzt das originale GTA-Radio

Blockiert nicht die Fahrzeugsteuerung

Features

Radio an/aus

Nächste / vorherige Station wechseln

HUD mit Sendername + Hintergrundbild

Automatisches Scrollen bei langen Sendernamen

Original Radio-HUD wird ausgeblendet

Nutzung auf dem Server

Resource in den Serverordner packen (z. B. resources/radio)

In der server.cfg aktivieren:

ensure radio


Hotkeys im Spiel nutzen:

Funktion	Taste
Radio an/aus	NUMPAD0
Nächste Station	NUMPAD*
Vorherige Station	NUMPAD/

Das HUD erscheint automatisch, wenn das Radio eingeschaltet wird, und verschwindet, wenn es ausgeschaltet wird oder das Fahrzeug gewechselt wird.
